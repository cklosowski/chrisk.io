If you know me even in the slightest, you know I'm a bit of a movie nerd. [Exhibit A](https://chrisk.io/trilogies-order-preference/). Being that the Back to the Future trilogy is one of my favorites, I was pumped to see that [Iain Poulson](http://www.polevaultweb.com/) created an awesome little JavaScript tool called Delorean Ipsum, which generates 'lorem ipsum' type text, from the script of Back to the Future. Yeah, legit right?!

![Screen Shot 2016-06-20 at 10.34.22 PM](/images/blog/Screen-Shot-2016-06-20-at-10.34.22-PM-1024x838.png)

I've been using Delorean Ipsum for a while now, to fill in my development environments when I need to write a page, post, or product content. Why not just use Lorem Ipsum? Well, it's freeking boring! I love opening up my development environment to see quotes like "Great Scott!" and "There's that word again, heavy. Why are things so heavy in the future. Is there a problem with the Earth's gravitational pull?"

Anyway, I wanted a quicker way to add this content to my WordPress post editor than visiting [deloranipsum.com](http://deloranipsum.com) every time I needed it...so I built a little plugin.

#### WordPress Delorean Ipsum Generator

This is a pretty simple plugin, [available on GitHub](https://github.com/cklosowski/wp-delorean-ipsum) if you want to look it over or suggest changes. Using the button below, you can quickly download a .zip file that you can install on your WordPress site as well. It adds a little 'Delorean Ipsum' button to your post edit screen, and provides you with a quick configuration and button to insert the text into the post editor. Nothing fancy, just quick and simple, for when you want a less boring development site.
