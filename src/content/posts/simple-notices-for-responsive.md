I've been using it for quite some time, but the other day I just pushed a new plugin to Github that users of the "[Responsive](http://wordpress.org/themes/responsive)" WordPress theme might find useful. It's called "[Simple Notices for Responsive](https://github.com/cklosowski/simple-notices-for-responsive)" and I'd love it if you'd take it for a spin on your Responsive powered website.  
  
Originally I wrote it for [WP-Push.com](https://wp-push.com), so I could announce new plugins or discounts. I figured it was useful enough to give to the masses. Since it's initial push, I've added the color picker and a live updated preview to the admin settings. Here are some screenshots...
